# Changelog

## 1.0.2

- Version aligned with the Channels DVR (Intel / AMD) app so both share the
  same version number. No functional change.

## 1.0.0

- Initial release. Channels DVR Server with NVIDIA NVENC hardware transcoding,
  wrapping the official `fancybits/channels-dvr:nvidia` image. Requires a host
  with NVIDIA drivers and GPU device access (see the documentation).
